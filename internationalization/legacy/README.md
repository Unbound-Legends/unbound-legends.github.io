# Legacy translation catalogs

These files are exact snapshots of the public catalogs at commit `0e1c441d7ee3f6797e9152382d70051856309ee1`.

They are preserved for compatibility, attribution research, and manual human recovery. Their keys do not match the current RPGSessions catalog, and some historical target files contain English fallback text. Never import or pretranslate from them automatically.

The original hosted paths remain in place for unknown consumers. [`manifest.json`](manifest.json) records every compatibility path, archive path, and SHA-256 checksum. The verifier also compares each snapshot with the file stored at the recorded source commit.
